## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Personal-assistant-Amanda.

```bash
speech:
pip install SpeechRecognition

answer:
pip install pyttsx3
pip install playsound
pip install opencv-python
```
